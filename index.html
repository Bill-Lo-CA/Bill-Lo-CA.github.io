Featured · Chromium Extension · Local AI Backend

ContextBubble

Functional prototype

A Chromium extension that prepares timestamped learning notes for YouTube videos and displays reviewed concept bubbles near the relevant playback moment.

Problem

Video learners often need concept explanations at the exact moment a term appears, without leaving the page.

Built

Caption-first ingestion, whisper.cpp ASR fallback, persistent preparation jobs, concept/reviewer stages, validation, caching, pairing, and overlay rendering.

Decision

Kept the backend local-first and separated deterministic testing, Gemini/Ollama analysis, and translation providers.

Status

End-to-end prototype works with explicit demo fixtures and live caption/ASR paths; settings and learning features continue to evolve.

Featured · SvelteKit · FastAPI · Local LLM

SummerDay

Vertical slice implemented

A local-first French learning application that creates one short reading lesson and a focused vocabulary set each day.

Problem

Daily language study needs content that is short, level-aware, source-grounded, and supported by useful pronunciation audio.

Built

Vikidia retrieval, Stanza French NLP, A1 suitability signals, Ollama structured generation, immutable audio packages, review gates, and publish commands.

Decision

Separated generation, audio, review, and publish stages so failures can resume without replacing the selected source article.

Status

The first daily-lesson vertical slice is implemented with model-independent tests; accounts, ASR scoring, and scheduling remain future work.
